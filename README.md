# Flappy-Bird-Arcade
копия аркадной игры Flappy bird на основе языка python и библиотеки Arcade
проект будет повторять концепцию оригинальной игры Flappy Bird, цель которой проходить проходить бесконечную, генерируемую случайным образом полосу препятствий и набрать максимальное колличество очков
препятсвия представляют из себя столбы длинной от верхнего края экрана до нижнего и имеют зазор в случайном месте, в который нужно попасть персонажу
персонаж представляет из себя птицу, которая бесконечно пикирует и набирает высоту за счёт коротких прыжков 
при касании края экрана или препятсвия игра оканчивается и выводится счёт игры
игрок получает 1 окчо счёта за удачное прохождение через препятсвие
