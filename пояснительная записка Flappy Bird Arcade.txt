ПОЯСНИТЕЛЬНАЯ ЗАПИСКА
к курсовому/учебному проекту
Название проекта: Flappy Bird – классическая аркадная игра с элементами настроек и главного меню
Цель проекта
Разработка полноценной 2D-аркадной игры в жанре «бесконечный раннер» на основе классической игры Flappy Bird с использованием библиотеки Arcade.
Основные задачи, которые были решены:

Создание главного меню с тремя кнопками (Играть, Настройки, Выход)
Реализация диалогового окна подтверждения выхода с затемнением фона
Разработка игрового экрана с управлением персонажем (прыжок по клику ЛКМ или пробел)
Генерация и движение препятствий (труб) с случайным зазором
Подсчёт очков при прохождении через трубы
Экран окончания игры с отображением счёта и кнопками перезапуска / возврата в меню
Реализация меню настроек с сохранением параметров в файл
Применение сохранённых настроек к игровому процессу (сложность, скин персонажа, громкость)

Использованные технологии и библиотеки

Язык программирования: Python 3
Основная библиотека для графики и ввода: Arcade 3.x
Графический интерфейс (меню, кнопки, слайдеры): arcade.gui
Сохранение настроек: JSON (модуль json)
Работа с файлами: встроенные средства Python (open, json.load/dump)

Структура проекта
textFlappyBird/
├── main.py               — точка входа, создание окна и запуск
├── mainmenu.py           — главное меню и диалог выхода
├── settings.py           — меню настроек (сложность, громкость, скин)
├── gamescreen.py         — основной игровой экран
├── styles.py             — общие стили кнопок (импортируется в меню)
└── settings.txt          — файл сохранения настроек (создаётся автоматически)
Основные реализованные механики

Главное меню
Три кнопки: «ИГРАТЬ», «НАСТРОЙКИ», «ВЫХОД»
Подтверждение выхода через модальное окно с затемнением фона

Игровой процесс
Управление: ЛКМ или ПРОБЕЛ — прыжок
Гравитация и физика падения
Генерация пар труб с случайным зазором
Подсчёт очков при прохождении через зазор
Game Over при столкновении с трубой или краем экрана

Меню настроек
Выбор сложности: Легко / Средне / Сложно
Регулировка громкости (слайдер 0–100%)
Выбор скина персонажа (робот / птица / самолёт)
Все параметры сохраняются в файл settings.txt в формате JSON

Применение настроек в игре
Сложность влияет на силу гравитации и частоту появления труб
Скин меняет внешний вид персонажа
Громкость (пока выводится в консоль при запуске игры)


Особенности реализации

Полностью объектно-ориентированный подход (каждый экран — отдельный класс-наследник arcade.View)
Корректное переключение между экранами (меню ↔ игра ↔ настройки ↔ game over)
Сохранение и загрузка настроек без потери данных при перезапуске
Использование встроенных ресурсов Arcade (:resources:) для спрайтов
Обработка событий мыши и клавиатуры через встроенные методы Arcade

Возможные улучшения (на перспективу)

Добавление звуковых эффектов (прыжок, столкновение, набор очков)
Реализация настоящей громкости (музыка + звуки через arcade.Sound)
Анимация крыльев персонажа
Таблица рекордов (сохранение в файл)
Фон с параллакс-эффектом
Кнопки на экране game over вместо текста

Вывод
Проект представляет собой полноценную мини-игру с современным графическим интерфейсом, сохранением настроек и базовой прогрессией сложности.
Работа демонстрирует навыки работы с библиотекой Arcade, ООП, обработкой событий, GUI и файловым вводом/выводом.
