## Техническое задание: игра *Flappy Bird* на Python (библиотека Arcade)

### 1. Общие сведения

**Название проекта:** Flappy Bird (клон классической версии).  
**Платформа:** десктоп (Windows/macOS/Linux).  
**Язык программирования:** Python 3.8+.  
**Графический фреймворк:** Arcade (версия ≥ 2.6).  
**Цель:** реализовать функционально полный клон игры *Flappy Bird* с сохранением механики, стилистики и уровня сложности оригинала.

### 2. Функциональные требования

#### 2.1. Основные механики
- **Управление птицей:**  
  - нажатие клавиши **(Пробел / ЛКМ)** — птица делает «взмах» и поднимается;  
  - без ввода — птица падает под действием гравитации.  
- **Гравитация:** постоянное ускорение вниз ($a = 0{,}5\ \text{px/frame}^2$).  
- **Столкновения:**  
  - с трубами — игра заканчивается;  
  - с землёй/верхней границей экрана — игра заканчивается.  
- **Счёт:** +1 очко за каждую пройденную пару труб.  
- **Экран конца игры:** отображение итогового счёта и предложения начать заново (клавиша **R** или клик).

#### 2.2. Визуальные элементы
- **Птица:** анимированный спрайт (2–3 кадра взмаха).  
- **Трубы:** пары вертикальных труб (верхняя и нижняя) с случайным зазором.  
- **Фон:** параллакс-скроллинг (земля, небо, далёкие объекты).  
- **Счёт:** отображение в верхнем левом углу ($\text{Score: } N$).  
- **Экран старта:** надпись «Нажмите Пробел» в центре.  
- **Экран конца игры:** «Game Over\nScore: $N$\nPress R to restart».

#### 2.3. Геймплей
- **Генерация труб:**  
  - первая труба появляется через $2{,}5\ \text{с}$ после старта;  
  - интервал между трубами: $1{,}5\ \text{с}$;  
  - вертикальный зазор: $100{-}150\ \text{px}$ (случайный).  
- **Скорость движения:**  
  - трубы движутся влево со скоростью $3\ \text{px/frame}$;  
  - фон движется медленнее ($1\ \text{px/frame}$).  
- **Рестарт:** по клавише **R** или клику — сброс счёта и позиции.

### 3. Технические требования

#### 3.1. Структура проекта
```
/flappy_bird/
  main.py              # точка входа
  settings.py          # константы (размеры, цвета, скорости)
  sprites/             # папки со спрайтами
    bird/
    pipes/
    background/
  sounds/              # звуковые эффекты (взмах, столкновение, очко)
  fonts/               # шрифт для текста
```

#### 3.2. Ресурсы
- Спрайты:  
  - птица (3 кадра, размер $34 \times 24\ \text{px}$);  
  - трубы (размер $52 \times 320\ \text{px}$, зелёный цвет);  
  - земля ($400 \times 56\ \text{px}$);  
  - фон ($400 \times 600\ \text{px}$).  
- Звуки:  
  - `flap.wav` — взмах;  
  - `hit.wav` — столкновение;  
  - `score.wav` — очко.  
- Шрифт: `arcade.Font` (стандартный или кастомный `.ttf`).

### 4. Этапы разработки

1. **Настройка окружения**  
   - Установка Python и Arcade: `pip install arcade`.  
   - Создание структуры проекта.

2. **Базовая отрисовка**  
   - Окно игры ($400 \times 600$).  
   - Фон и земля (статично).  
   - Спрайт птицы в центре.

3. **Механики движения**  
   - Гравитация и управление взмахом.  
   - Анимация птицы.

4. **Трубы и коллизии**  
   - Генерация труб с интервалом.  
   - Проверка столкновений (AABB).  
   - Подсчёт очков.

5. **UI и звуки**  
   - Отображение счёта.  
   - Экран конца игры.  
   - Подключение звуков.

6. **Тестирование и оптимизация**  
   - Проверка на баги (прохождение сквозь трубы, некорректный счёт).  
   - Настройка баланса сложности.

### 5. Критерии приёмки

- Игра запускается без ошибок.  
- Птица реагирует на ввод и подчиняется гравитации.  
- Трубы генерируются с правильным интервалом и зазором.  
- Счёт увеличивается при прохождении труб.  
- При столкновении — экран конца игры с итоговым счётом.  
- Рестарт по клавише **R** работает корректно.  
- Визуальный стиль соответствует оригиналу (цвета, спрайты).  
- FPS стабилен ($60$).
